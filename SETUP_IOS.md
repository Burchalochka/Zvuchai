# Инструкция по настройке iOS для проекта

## Шаг 1: Настройка Xcode

Откройте терминал и выполните эти команды по очереди:

```bash
# 1. Переключить xcode-select на полный Xcode
sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer

# 2. Принять лицензию Xcode
sudo xcodebuild -license accept

# 3. Перейти в папку ios
cd ios

# 4. Установить зависимости CocoaPods
pod install

# 5. Вернуться в корень проекта
cd ..

# 6. Запустить приложение на iOS симуляторе
npx react-native run-ios
```

## Важно про пароль:

- **Пароль для `sudo`** - это пароль вашего пользователя macOS (тот, которым вы входите в Mac при включении)
- Это **НЕ** пароль Apple ID
- Это **НЕ** пароль iCloud
- Это пароль, который вы вводите при входе в систему macOS

## Если не помните пароль:

1. Перезагрузите Mac
2. При загрузке зажмите **Command + R** (удерживайте до появления логотипа Apple)
3. В Recovery Mode выберите **Utilities** → **Terminal**
4. Выполните команду: `resetpassword`
5. Следуйте инструкциям для сброса пароля

## После настройки:

Приложение автоматически откроется в iOS симуляторе, и вы сможете протестировать голосовую запись на iPhone!
